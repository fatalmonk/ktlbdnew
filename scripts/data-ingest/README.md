# Archived Scripts

This directory contains one-off scripts and tools that are not part of the main KTL website project.

## Contents

- `scrape_celebpink.py` - Python script for scraping product data from celebpink.com
- `scrape_celebpink_playwright.py` - Playwright-based scraper for celebpink.com
- `download_images.py` - Script to download product images
- `download_logos.py` - Script to download logo assets
- `download_logos_highres.py` - Script to download high-resolution logos

## Note

These scripts were used for data collection/research purposes and are not required for the main website functionality. They are kept here for reference only.
