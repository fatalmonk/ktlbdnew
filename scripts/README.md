# Scripts

Utility scripts and automation tools for the KTL Website project.

## Data Ingestion

- `data-ingest/` - Scripts for scraping and data collection
  - Python scripts for web scraping (Playwright-based)
  - Image download utilities
  - Logo extraction tools

## Usage

Most scripts are one-off utilities. See individual script files for usage instructions.

## Dependencies

- Python 3.x
- Playwright for Python
- aiohttp for async HTTP requests
